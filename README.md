# PixelFlow Image Optimizer & Resizer

A fast, secure, and easy-to-use web tool to resize, compress, and convert bulk images to PDF directly in your browser. 

Because it runs entirely on the client-side, your images are never uploaded to any server, guaranteeing 100% privacy.

## ✨ Features

* **Bulk Processing:** Drag and drop multiple images at once.
* **Custom Resizing:** Set a max width or keep the original size.
* **Smart Compression:** Easily shrink file sizes to save space.
* **PDF Generator:** Combine all your processed images into a single A4 PDF document.
* **Zero Installation:** Works offline right in your browser.

## 🚀 How to Use

You don't need to install any software or dependencies to use this tool:

1. Download or clone this repository.
2. Double-click the `index.html` file to open it in Chrome, Safari, Edge, or Firefox.
3. Drag your images in, adjust your settings, and click "Process"!

## 🛠️ Built With

* HTML5
* CSS3 (Vanilla)
* JavaScript
* [jsPDF](https://github.com/parallax/jsPDF) (For PDF generation)
